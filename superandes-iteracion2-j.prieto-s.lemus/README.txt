Instrucciones de instalación de aplicación en el IDE Eclipse:

1) Descargar y guardar la carpeta del proyecto en workspace
2) Dirigirse a: 
	- File -> Import -> General -> Existing Projects into Workspace -> <Ingresar ruta donde está el proyecto>
3) Dirigirse dentro de la carpeta principal del proyecto a:
	- uniandes.isis2304.superandes.interfaz -> InterfazSuperandes.java
4) Correr el archivo InterfazSuperandes.java
5) Ingresar o Registrar el Id de un cliente dentro de la aplicación para probar todas las funcionalidades
